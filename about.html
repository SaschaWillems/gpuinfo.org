About gpuinfo.org

This is the launchpad for my GPU hardware databases targeting the open graphics API of Khronos. They're aimed at developers that need to know what features are supported by a driver/GPU combination or to check how widespread certain features are.
These databases and the tools to fill it are developed and maintained by me (Sascha Willems) in my spare time.
No profit is made from this data, nor is this data used in any commercial way and no personal data is transferred, stored or passed.

If you want to contribute to the development, you can find the source code at https://github.com/SaschaWillems.

Donating

All of my tools and the databases itself are free-to-use, open source and hosted by me free of charge, feel free to donate ;)
